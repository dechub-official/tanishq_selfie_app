# Change Summary: View Data Endpoint Modified to Show Attendees

## Date: January 30, 2026

## Overview
The `/events/getinvitedmember` endpoint has been successfully modified to display **attendees data** instead of invitees data.

---

## Changes Made

### 1. Service Layer Changes
**File:** `TanishqPageService.java`

**Before:**
```java
public List<?> getInvitedMember(String eventId) {
    try {
        List<Invitee> invitees = inviteeRepository.findByEventId(eventId);
        return invitees.stream().map(invitee -> {
            Map<String, Object> map = new HashMap<>();
            map.put("name", invitee.getName());
            map.put("contact", invitee.getContact());
            return map;
        }).collect(Collectors.toList());
    } catch (Exception e) {
        return new ArrayList<>();
    }
}
```

**After:**
```java
public List<?> getInvitedMember(String eventId) {
    try {
        List<Attendee> attendees = attendeeRepository.findByEventId(eventId);
        return attendees.stream().map(attendee -> {
            Map<String, Object> map = new HashMap<>();
            map.put("name", attendee.getName());
            map.put("contact", attendee.getPhone());
            map.put("like", attendee.getLike());
            map.put("firstTimeAtTanishq", attendee.getFirstTimeAtTanishq());
            return map;
        }).collect(Collectors.toList());
    } catch (Exception e) {
        return new ArrayList<>();
    }
}
```

**Key Changes:**
- Changed data source from `inviteeRepository` to `attendeeRepository`
- Changed entity type from `Invitee` to `Attendee`
- Changed field name from `getContact()` to `getPhone()`
- **Added** `like` field to the response
- **Added** `firstTimeAtTanishq` field to the response

---

## Data Source Changes

### Previous Data Source
- **Table:** `invitees`
- **Fields Returned:** 
  - `name` - Invitee's name
  - `contact` - Invitee's phone number

### New Data Source
- **Table:** `attendees`
- **Fields Returned:**
  - `name` - Attendee's name
  - `contact` - Attendee's phone number
  - `like` - Whether they liked the event (Yes/No)
  - `firstTimeAtTanishq` - Whether this is their first visit (true/false)

## Impact

### Frontend Display
The "View Data" button now shows:
1. **Serial Number (Sno)** - Generated by frontend
2. **Customer Name** - From `attendees.name`
3. **Customer Phone** - From `attendees.phone`
4. **Like** - From `attendees.like` (Yes/No)
