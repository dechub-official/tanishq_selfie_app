🚀 HOW TO USE (3 EASY STEPS):

STEP 1: Setup Production Database
──────────────────────────────────
On your production server, run:

  Windows:  SETUP_PRODUCTION_DATABASE.bat

  OR manually:
  mysql -u root -p
  CREATE DATABASE selfie_prod CHARACTER SET utf8mb4;


STEP 2: Build for Pre-Production
──────────────────────────────────
  Double-click:  BUILD_PREPROD.bat

  This will:
  ✓ Use preprod configuration
  ✓ Connect to selfie_preprod database
  ✓ Create WAR for port 3000


STEP 3: Build for Production
──────────────────────────────
  Double-click:  BUILD_PROD.bat

  This will:
  ✓ Use production configuration
  ✓ Connect to selfie_prod database
  ✓ Create WAR for port 3001
